# Backup Agent Releases

This repository contains signed binary releases of the Milbo Backup Agent.

**Source Code**: [github.com/Milbo-GmbH/milbo-backup-tool](https://github.com/Milbo-GmbH/milbo-backup-tool) (private)

## Download

Download the latest release for your platform from the [Releases](https://github.com/Milbo-GmbH/backup-agent-releases/releases) page.

## Verification

All binaries are signed with Ed25519. Signatures are included in release assets (`.sig` files).

## Installation

See [Installation Guide](https://github.com/Milbo-GmbH/backup-agent-releases/Installation) for platform-specific instructions.

## Supported Platforms

- Linux (amd64, arm64)
- macOS (amd64 Intel, arm64 Apple Silicon)
- Windows (amd64)

## Release Schedule

Releases follow semantic versioning: `vMAJOR.MINOR.PATCH`
